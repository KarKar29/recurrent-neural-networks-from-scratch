# Recurrent Neural Networks from Scratch

A NumPy implementation of Recurrent Neural Networks (RNNs) and Long Short-Term Memory (LSTM) networks built entirely from scratch without using deep learning frameworks such as TensorFlow or PyTorch.

---

## Overview

This project implements the forward propagation of vanilla Recurrent Neural Networks (RNNs) and Long Short-Term Memory (LSTM) networks using only Python and NumPy.

The implementation demonstrates how sequential data is processed over time while maintaining hidden states and memory cells for sequence modeling tasks.

---

## Features

- Vanilla RNN Cell
- Multi-step RNN Forward Pass
- LSTM Cell
- Multi-step LSTM Forward Pass
- Softmax Output Layer
- Sigmoid Activation
- Tanh Activation
- Hidden State Updates
- Memory Cell Updates
- Vectorized NumPy Operations
